# Plugins Catalog — removed

The starter no longer installs Claude Code plugins via a marketplace
(`claude plugin marketplace add` / `claude plugin install`). **All skills are now
installed with `bunx skills add <repo-url> --skill <name>`** — see
[`skills-catalog.md`](./skills-catalog.md).

This file is intentionally kept as a redirect and is not referenced by `SKILL.md`. It
can be deleted.
